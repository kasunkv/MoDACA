RegEx for Data Validation
-------------------------

Validating:

* Names 			=> /^[A-Z][A-Za-z. ]+$/
* Gender			=> /^(Male|Female)$/
* Contact No (SL)	=> /^0[1-9][0-9]{8}$/
* Address			=> /^[0-9A-Za-z\-, ./()]+$/
* Designations 		=> /^[A-Za-z\-, ./()]+$/
* Username			=> /^[A-Za-z0-9_\-]{4,40}$/
* Password			=> /^([\w\d\W\][^<>{}\[\]\(\)])+$/
* Long Text (bio) 	=> /([\w\d\W\s][^<>{}\[\]\(\)])+/
